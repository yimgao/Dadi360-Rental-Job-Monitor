## 六、行业周期位置判断

| 周期阶段 | 典型特征 | QRVO当前状态 |
|:---|:---|:---:|
| **繁荣期** | 营收/利润创新高，PE压缩，产能满载 | FY2022（已过） |
| **下行期** | 营收下滑20-30%，利润率崩塔，库存积压 | FY2023-FY2025（已过） |
| **底部/复苏前期** | 营收企稳，毛利率触底，FCF仍为正 | **FY2026（当前）** |
| **温和复苏期** | 营收恢复增长，利润率改善，产能利用率回升 | **FY2027预期** |
| **繁荣期** | 周期顶峰，高利润+高PE | 预计FY2028+ |

**周期参考：
- 射频半导体周期通常2-3年一轮
- 本轮下行自FY2022末~FY2023初开始，持续约3年
- **当前（FY2026）大概率处于周期底部至复苏初期**
- 毛利率从FY2025的41.3%修复到45.9%，趋势已确认改善
- 若5G渗透率 + 物联网 + 苹果份额增长共振，FY2027-FY2028可能迎来新的景气周期

## 七、综合评估与巴菲特视角建议

| 维度 | 评价 |
|:---|:---:|
| **护城河** | 中等。射频前端（BAW/SAW滤波器）技术壁垒高，苹果/三星供应链粘性强，但客户集中度风险（苹果占~30%+） |
| **财务稳健性** | 优秀。净债务极低、FCF持续为正、资产负债率<45% |
| **盈利能力** | 一般。ROE 10%刚过及格线，周期顶峰可达22%，底部为负 |
| **估值安全边际** | 当前Forward PE 12x，中性估值$109有17%上行 |
| **管理层资本配置** | 一般。多次高价收购（商誉$23.5亿），股票回购时机有待改进 |
| **行业周期性** | 射频行业属于温和周期型，非强周期，底部可适当逆向布局 |

**最终判断**：
Qorvo在周期底部恢复初期，财务极其稳健（净债务仅$3.3亿），FCF强劲（~$6.8亿/年）。当前**Forward PE 12x**提供了较好的安全边际。对愿意承受周期波动（20-25%下行风险）的价值投资者，$90以下具备分批建仓价值。核心观察点：FY2027毛利率能否站穈48%+、苹果业务份额变化。

**建议买入区间**：$75-90（安全边际充足）
**中性持有区间**：$90-110
**偏高估区间**：$130+（接近乐观情景目标）
